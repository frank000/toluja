# Toluja Go Landing (WordPress)

Landing page comercial para o **Toluja Go** com foco em aumento de alcance e vendas.

## O que já está pronto
- Hero comercial com narrativa de valor
- Seção de benefícios orientada a conversão
- Planos:
  - Grátis até vender **1.500** na plataforma
  - **R$ 59,90** plano básico
  - **R$ 79,90** com customizações e sem anúncios
- Formulário de cadastro para teste gratuito
- Salvamento de leads no painel WordPress (`Leads Toluja`)
- Envio de notificação por e-mail para o admin do site

## Como instalar
1. Copie a pasta `toluja-go-landing` para `wp-content/themes/`.
2. No WordPress, vá em `Aparência > Temas` e ative `Toluja Go Landing`.
3. Em `Configurações > Leitura`, selecione a opção de página inicial estática apontando para a home (o tema usa `front-page.php`).

## Arquivos principais
- `style.css`: identidade visual e responsividade
- `front-page.php`: estrutura da landing
- `functions.php`: processamento do formulário e registro dos leads
