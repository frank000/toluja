# One-Click Windows Installer

Arquivos desta pasta:

- `Instalar Impressora Toluja.bat` (arquivo para o cliente clicar)
- `install-windows.ps1` (instalador robusto)

O script instala o agente em `C:\Program Files\Toluja\PrintAgent` e cria o servico `TolujaPrintAgent`.
A configuracao fica em `config.json` (na mesma pasta da instalacao).
